# GLTF/GLB

glTF (.gltf) → 3D model + external files

GLB (.glb) → everything packed into one file

| Feature        | glTF (.gltf)         | GLB (.glb)     |
| -------------- | -------------------- | -------------- |
| File type      | JSON text            | Binary         |
| Textures       | External image files | Embedded       |
| Size           | Multiple files       | Single file    |
| Human-readable | ✅ Yes               | ❌ No          |
| Web usage      | Less common          | ⭐ Most common |
