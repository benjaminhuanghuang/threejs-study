# Set up a three.js project with parcel

install parcel
```
    npm i parcel -g
```

Install  three.js
```
    npm i three
```

Use three.js
```js
    import * as THREE from 'three';
```

Use parcel to run the project
```html
   <script src="./app.js"></script>
```


Run 
```
    parcel index.html
```
