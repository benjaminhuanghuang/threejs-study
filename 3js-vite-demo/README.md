# Three.js Crash Course For Beginners | Create This Awesome 3D Website!

https://www.youtube.com/watch?v=_OwJV2xL8M8&t=777s

## Create project

```bash
    npm create vite@latest

    npm init vite@latest demo
```

## Install three.js

```bash
    npm i three gasp
```

## Run

```bash
    npm run dev
```
